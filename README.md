# C# Task Manager

## Description
This project is a simple console-based task manager written in C#. Users can add tasks, view tasks, and remove completed tasks.

## Purpose
The purpose of this software is to practice C# programming fundamentals, including loops, methods, lists, conditionals, and user input.

## Features
- Add tasks
- View tasks
- Complete/remove tasks
- Interactive menu system

## Technologies Used
- C#
- .NET 10
- Visual Studio Code
- GitHub

## How to Run the Program
1. Open the project folder in Visual Studio Code
2. Open the terminal
3. Run the command:

```bash
dotnet run
