# Task Manager

## Overview

The Task Manager is a console-based application written in C#. This software allows users to create, view, complete, and delete tasks. The purpose of this project is to practice C# programming concepts including lists, loops, methods, conditionals, and user input.

This project was developed for the CSE 310 Applied Programming course.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Exit the application
